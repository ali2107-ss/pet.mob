## 📋 ШАГ 1: Проверьте структуру таблиц (2 минуты)

### Выполните скрипт проверки:

1. Откройте Supabase → **SQL Editor**
2. Скопируйте файл **`check_tables_structure.sql`**
3. Вставьте и нажмите **RUN**
4. Посмотрите результаты

### Что проверяем:
- Структуру таблицы `products` (должны быть колонки: id, name, description, price, category, image_url)
- Структуру таблицы `cart_items` (должны быть: user_id, product_id, quantity)
- Структуру таблицы `favorites` (должны быть: user_id, product_id)
- Наличие RLS политик

## ✅ Как работает (БЕЗ локального хранилища):

### Корзина:
1. Пользователь добавляет товар → сразу сохраняется в Supabase `cart_items`
2. При открытии приложения → загружается из Supabase
3. При выходе из аккаунта → корзина очищается

### Избранное:
1. Пользователь добавляет в избранное → сразу сохраняется в Supabase `favorites`
2. При открытии приложения → загружается из Supabase
3. При выходе из аккаунта → избранное очищается

### Все данные хранятся ТОЛЬКО в Supabase! 🎯

## 🔧 Если возникают ошибки:

### ❌ Ошибка: "relation does not exist"
**Решение:** Таблица не создана. Выполните `fix_cart_favorites_tables.sql`

### ❌ Ошибка: "permission denied"
**Решение:** Проблема с RLS. Выполните `fix_cart_favorites_tables.sql` - там настроены политики

### ❌ Корзина не сохраняется
**Решение:** 
1. Проверьте, что пользователь авторизован
2. Откройте Supabase → Table Editor → cart_items
3. Проверьте, есть ли записи
4. Если нет - проверьте RLS политики

### ❌ Товары не загружаются
**Решение:** Добавьте тестовые товары (см. ШАГ 3)

## 📊 Проверка через Supabase Dashboard

### Проверка корзины:
```sql
-- Посмотреть все корзины
SELECT 
  cart_items.user_id,
  cart_items.product_id,
  cart_items.quantity,
  products.name,
  products.price
FROM cart_items
LEFT JOIN products ON cart_items.product_id = products.id;
```

### Проверка избранного:
```sql
-- Посмотреть все избранное
SELECT 
  favorites.user_id,
  favorites.product_id,
  products.name
FROM favorites
LEFT JOIN products ON favorites.product_id = products.id;
```
